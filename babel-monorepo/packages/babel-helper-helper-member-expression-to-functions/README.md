# @babel/babel-helper-helper-member-expression-to-functions

> Babel babel-helper package for transforming JavaScript

## Install

```bash
npm install @babel/babel-helper-helper-member-expression-to-functions
```

## Usage

```javascript
import * as transformer from "@babel/babel-helper-helper-member-expression-to-functions";

transformer.transform(code, options);
```
