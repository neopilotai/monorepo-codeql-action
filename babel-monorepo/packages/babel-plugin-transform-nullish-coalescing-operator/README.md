# @babel/babel-plugin-transform-nullish-coalescing-operator

> Babel babel-plugin-transform package for transforming JavaScript

## Install

```bash
npm install @babel/babel-plugin-transform-nullish-coalescing-operator
```

## Usage

```javascript
import * as transformer from "@babel/babel-plugin-transform-nullish-coalescing-operator";

transformer.transform(code, options);
```
