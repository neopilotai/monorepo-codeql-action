# @babel/babel-runtime-helpers-interopRequireWildcard

> Babel babel-runtime package for transforming JavaScript

## Install

```bash
npm install @babel/babel-runtime-helpers-interopRequireWildcard
```

## Usage

```javascript
import * as transformer from "@babel/babel-runtime-helpers-interopRequireWildcard";

transformer.transform(code, options);
```
