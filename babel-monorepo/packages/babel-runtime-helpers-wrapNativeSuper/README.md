# @babel/babel-runtime-helpers-wrapNativeSuper

> Babel babel-runtime package for transforming JavaScript

## Install

```bash
npm install @babel/babel-runtime-helpers-wrapNativeSuper
```

## Usage

```javascript
import * as transformer from "@babel/babel-runtime-helpers-wrapNativeSuper";

transformer.transform(code, options);
```
