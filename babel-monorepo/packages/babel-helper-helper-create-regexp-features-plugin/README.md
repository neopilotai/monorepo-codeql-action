# @babel/babel-helper-helper-create-regexp-features-plugin

> Babel babel-helper package for transforming JavaScript

## Install

```bash
npm install @babel/babel-helper-helper-create-regexp-features-plugin
```

## Usage

```javascript
import * as transformer from "@babel/babel-helper-helper-create-regexp-features-plugin";

transformer.transform(code, options);
```
