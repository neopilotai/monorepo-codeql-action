# @babel/babel-preset-env-preset-plugin-corejs

> Babel babel-preset-env package for transforming JavaScript

## Install

```bash
npm install @babel/babel-preset-env-preset-plugin-corejs
```

## Usage

```javascript
import * as transformer from "@babel/babel-preset-env-preset-plugin-corejs";

transformer.transform(code, options);
```
