# @babel/babel-plugin-transform-concise-method-property

> Babel babel-plugin-transform package for transforming JavaScript

## Install

```bash
npm install @babel/babel-plugin-transform-concise-method-property
```

## Usage

```javascript
import * as transformer from "@babel/babel-plugin-transform-concise-method-property";

transformer.transform(code, options);
```
