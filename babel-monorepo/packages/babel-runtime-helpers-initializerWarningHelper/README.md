# @babel/babel-runtime-helpers-initializerWarningHelper

> Babel babel-runtime package for transforming JavaScript

## Install

```bash
npm install @babel/babel-runtime-helpers-initializerWarningHelper
```

## Usage

```javascript
import * as transformer from "@babel/babel-runtime-helpers-initializerWarningHelper";

transformer.transform(code, options);
```
