# @babel/babel-helper-helper-named-capturing-groups-regex

> Babel babel-helper package for transforming JavaScript

## Install

```bash
npm install @babel/babel-helper-helper-named-capturing-groups-regex
```

## Usage

```javascript
import * as transformer from "@babel/babel-helper-helper-named-capturing-groups-regex";

transformer.transform(code, options);
```
