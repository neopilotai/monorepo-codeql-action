# @babel/babel-runtime-helpers-objectDestructuringEmpty

> Babel babel-runtime package for transforming JavaScript

## Install

```bash
npm install @babel/babel-runtime-helpers-objectDestructuringEmpty
```

## Usage

```javascript
import * as transformer from "@babel/babel-runtime-helpers-objectDestructuringEmpty";

transformer.transform(code, options);
```
