# @babel/babel-helper-helper-get-function-arity

> Babel babel-helper package for transforming JavaScript

## Install

```bash
npm install @babel/babel-helper-helper-get-function-arity
```

## Usage

```javascript
import * as transformer from "@babel/babel-helper-helper-get-function-arity";

transformer.transform(code, options);
```
